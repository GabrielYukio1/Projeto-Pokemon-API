# Projeto PokéAPI

Projeto desenvolvido para fins educacionais utilizando Node.js e a PokéAPI.

O objetivo deste projeto é praticar consumo de API, requisições HTTP e manipulação de dados em JavaScript.

## Funcionalidades
- Buscar um Pokémon pelo nome
- Exibir informações no terminal

## Tecnologias utilizadas
- Node.js
- JavaScript
- Axios
- PokéAPI

## Observação
Projeto para fins educacionais utilizando a PokéAPI.  
Todos os dados pertencem aos seus respectivos proprietários.
